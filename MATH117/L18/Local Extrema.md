*Key Concepts:*
___

- Def'n: For the interval $I = [a, c]$, the point x = b where $b\in I$ is a local minimum $\iff$ $f(x) < f(b)$ for all x $\in I$ and there exists $x_1$ and $x_2$ s.t $x_{1} < b < x_{2}$. 
	- Condition 1: $f(x) < f(b)$ for all x $\in I$
	- Condition 2: There exists $x_1$ and $x_2$ s.t $x_{1} < b < x_{2}$. 
- Def'n: For the interval $I = [a, c]$, the point x = b where $b\in I$ is a local maximum $\iff$ 
	- Condition 1: $f(x) > f(b)$ for all x $\in I$
	- Condition 2: There exists $x_1$ and $x_2$ s.t $x_{1} < b < x_{2}$. 

- Note: Local extrema cannot occur at endpoints in the closed interval due to Condition 2.
