___

- given there is a crit.p at x = c, check if $f'(x)$ changes sign at x = c. This can typically be done by finding $f'(c)$ for values close to c, but not after another crit.p.
	- -ve to +v: Local min
	- +ve to -ve: Local max
	- No change: Not an extrema.
